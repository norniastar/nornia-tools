# GitHub Release 模板

## v0.0.1

`Nornia Tools` 的首个公开版本。

在线体验：[tools.nornia.top](https://tools.nornia.top)

### 包含工具

- JSON 编辑器
- 时间戳转换
- UUID 生成器
- 文本代码对比
- 计算稿纸

### 版本特性

- 提供统一的开发者工具工作台首页
- 支持每个工具的 hash 路由直达
- 统一工具页结构、按钮样式和复制反馈
- 支持部分工具的短时草稿保留
- 文本代码对比基于 Monaco Diff Editor

### 当前技术栈

- React
- TypeScript
- Vite
- Tailwind CSS
- Monaco Editor

### 说明

- 当前版本聚焦本地可直接完成的开发辅助能力
- 后续版本将继续按现有架构规范和设计规范扩展新工具
